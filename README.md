# SwasthyaAI
### AI‑Powered Early Health Risk Detection & First Aid Assistance System
**Tagline:** Detect Early. Act Fast. Save Lives.

---

## 1. Introduction
SwasthyaAI is an AI-powered healthcare assistant that analyzes user symptoms from text or images, predicts possible health conditions, detects severity level, provides first aid guidance, recommends the appropriate doctor, and supports mental health awareness.

---

## 2. Problem Statement
Many people cannot quickly understand the seriousness of their health symptoms. In emergencies, delayed first aid and wrong decisions can be dangerous. Mental health issues such as stress, anxiety, and depression are also common among teenagers and young adults but often ignored.

---

## 3. Solution
SwasthyaAI provides an AI-based system that:
- Analyzes symptoms from text or image
- Predicts possible conditions
- Detects severity level
- Provides first aid guidance
- Recommends doctor type
- Supports mental health awareness

---

## 4. Features
- Symptom → Condition AI Analysis
- Image-based disease detection
- Severity Detection (Safe / Attention / Critical)
- First Aid Instructions
- Doctor Recommendation + Map
- Mental Health Detection
- Mental Health Awareness Button
- AI Response Evaluation System
